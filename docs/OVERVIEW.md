# Project Overview

This `go-cache` project is intended to be a simple in-memory cache written in Golang. Currently, it serves as a basic Go module boilerplate and does not yet contain the actual cache implementation.

## Key Files

*   **`main.go`**: The primary entry point for the application. As of now, it prints "HELLO!" to the console. This file is where the core logic for the in-memory cache will eventually reside.
*   **`go.mod`**: Defines the Go module path (`calvinbrown085/go-cache`) and specifies the required Go version (`1.24.3`) for the project's dependencies and build environment.
*   **`README.md`**: Provides a high-level, human-readable description of the project's stated purpose as an in-memory cache.

## How to Run

To execute the current placeholder application:

1.  Ensure you have Go (version 1.24.3 or newer) installed on your system.
2.  Navigate to the root directory of the `go-cache` project in your terminal.
3.  Run the application using the Go command:
    ```bash
    go run main.go
    ```
    This command will compile and execute `main.go`, printing "HELLO!" to your console.
